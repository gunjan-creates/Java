# String and Exception Handling Package - 100 Beginner Java Programs

This package contains 100 example programs demonstrating String manipulation and exception handling.

## Topics Covered

### String Basics (Programs 1-25)
- String creation and immutability
- String methods (length, charAt, substring)
- String comparison
- String concatenation
- String formatting

### StringBuilder and StringBuffer (Programs 26-40)
- StringBuilder operations
- StringBuffer operations
- Performance comparison
- When to use each

### String Manipulation (Programs 41-60)
- String splitting
- String joining
- Regular expressions
- String replacement
- Case conversion

### Advanced String Topics (Programs 61-75)
- String pool
- Internationalization
- Character encoding
- String parsing

### Combined Topics (Programs 76-100)
- Exception handling with strings
- String validation
- Input parsing
- Error handling patterns

## How to Run

```bash
# Compile all programs
make build

# Run a specific program
make run CLASS=StringExceptionHandling.StringManupulationExample
make run CLASS=StringExceptionHandling.Program008
```

## Learning Path

1. Master basic String operations
2. Understand String immutability
3. Learn when to use StringBuilder
4. Practice with regular expressions
5. Combine with exception handling

## Tips for Beginners

- Strings are immutable in Java
- Use StringBuilder for multiple concatenations
- Understand String pool for memory efficiency
- Be careful with == vs .equals() for strings
- Use regex for complex pattern matching
