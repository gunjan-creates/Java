# Exception Handling Package - 100 Beginner Java Programs

This package contains 100 example programs demonstrating exception handling in Java.

## Topics Covered

### Basic Exception Handling (Programs 1-25)
- try-catch blocks
- Multiple catch blocks
- finally block
- Built-in exceptions
- Exception hierarchy

### Advanced Exception Handling (Programs 26-50)
- Custom exceptions
- throw and throws keywords
- Exception chaining
- Try-with-resources
- Multi-catch blocks

### Common Exception Types (Programs 51-75)
- NullPointerException
- ArrayIndexOutOfBoundsException
- NumberFormatException
- IOException
- ArithmeticException

### Best Practices (Programs 76-100)
- When to catch vs throw
- Exception handling patterns
- Logging exceptions
- Error recovery
- Exception anti-patterns

## How to Run

```bash
# Compile all programs
make build

# Run a specific program
make run CLASS=ExceptionHandelling.BuiltinException
make run CLASS=ExceptionHandelling.ExceptionExample008
```

## Learning Path

1. Understand exception basics and try-catch
2. Learn about different exception types
3. Study custom exceptions
4. Master best practices

## Tips for Beginners

- Always handle exceptions appropriately
- Don't catch Exception unless necessary
- Use specific exception types
- Clean up resources in finally or try-with-resources
- Log exceptions for debugging
