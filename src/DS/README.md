# Data Structures Package - 100 Beginner Java Programs

This package contains 100 example programs demonstrating fundamental data structures and algorithms in Java.

## Topics Covered

### Basic Data Structures (Programs 1-30)
- Arrays and array operations
- ArrayList usage
- LinkedList implementation concepts
- Stack operations
- Queue operations

### Hash-based Structures (Programs 31-50)
- HashMap usage and examples
- HashSet operations
- Hash table concepts
- Collision handling

### Advanced Structures (Programs 51-70)
- TreeMap operations
- TreeSet usage
- Priority Queues
- EnumMap examples

### Algorithms (Programs 71-100)
- Searching algorithms
- Sorting algorithms
- Basic algorithmic patterns
- Data manipulation

## How to Run

```bash
# Compile all programs
make build

# Run a specific program
make run CLASS=DS.Hashmapexample
make run CLASS=DS.Program004
```

## Learning Path

1. Start with array basics
2. Learn about hash-based data structures
3. Understand tree-based structures
4. Practice with algorithms

## Tips for Beginners

- Understand time and space complexity
- Compare different data structures for different use cases
- Practice implementing basic structures from scratch
- Use built-in Java collections for production code
